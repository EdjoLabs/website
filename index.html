blockchain is a decentralised, distributed, immutable and transparent way to store information online. edjo uses a custom blockchain based database to record content creator’s work on the network in order to ensure that copyright protections are established and maintained.

our ai web crawler includes audio and image recognition. using state-of-the-art machine learning algorithms, it is able to scan through the world-wide-web and report detected copyright infringements back to the user.
